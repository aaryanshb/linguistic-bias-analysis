# linguistic-bias-analysis
Analyses linguistic bias and power dynamics in media discourse using NLP and dependency parsing; developed metrics to quantify sentiment, agency, and framing across social groups, supported by interactive visualisations.

## Methods
- NLP (spaCy)
- n-grams
- sentiment analysis

## Goal
Identify bias in tone, framing, and agency.
